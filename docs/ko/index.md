안녕하세요, Neil Yan입니다. 블록체인과 Web3에 집중하는 프리랜서 개발자입니다.

**나는 무엇을 하는가**

2018년부터 블록체인 기술 개발에 참여해 왔습니다. 저는 분산화 기술의 가능성을 믿으며 Web3가 인터넷 가치 체계를 재구성할 것입니다. 여러 Web3 DApp 제품을 성공적으로 출시했으며 수천 명의 사용자에게 서비스를 제공하고 있습니다.

**기술 전문성**

- **스마트 계약 개발**: Solidity, Vyper, Hardhat과 Foundry 사용

- **Web3 통합**: Ethereum, Polygon, BNB Chain, Solana, Avalanche, Arbitrum, Optimism 지원
- **백엔드 서비스**: Node.js, Java, Python, MySQL, Redis
- **월렛 통합**: MetaMask, WalletConnect, Coinbase Wallet, Rabby
- **AI 통합**: Gemini API, OpenAI, 알리클라우드 통익천문
